#### simpleMSSpeech
A simple console application for MS Speech TTS.


#### REQUIRED
windows 10 x86_64
Visual Studio 2017

###### MicrosoftSpeechPlatformSDK_x64
The Microsoft Speech Platform SDK.

###### MSSpeech_TTS_ja-JP_Haruka

Voice package for jpanese language. For other language ,see https://www.microsoft.com/en-us/download/details.aspx?id=27224.

#### INSTALLATION

step 4: Press win + r to run 'ms-settings:easeofaccess-narrator' to select 'Narrator' for your windows
